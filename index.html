Tentang Saya

Saya adalah mahasiswa Teknik Informatika yang memiliki fokus kuat pada pengembangan web dan teknologi informasi modern. Saya terbiasa membangun aplikasi web responsif dengan struktur kode yang bersih dan terorganisir menggunakan HTML, CSS, dan JavaScript. Saya juga memahami dasar keamanan aplikasi web, termasuk pencegahan SQL Injection, XSS, serta penerapan validasi dan sanitasi input untuk memastikan sistem yang saya kembangkan aman dan andal.

Selain pengembangan web, saya memiliki minat besar dalam keamanan informasi dan jaringan komputer sebagai bagian dari penguatan kompetensi di bidang IT. Saya telah menyelesaikan sertifikasi KKNI Level II Teknik Komputer dan Jaringan serta terlibat dalam proyek perancangan UI/UX aplikasi kasir sebagai bagian dari pengembangan kemampuan analitis dan problem-solving saya. Saya terus mengembangkan keterampilan teknis dan siap berkontribusi dalam lingkungan profesional yang menantang.
